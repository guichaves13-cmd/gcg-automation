ℹ️ Como funciona

1️⃣ Upload do vídeo original (MP4)
2️⃣ Extração do áudio e primeiro frame
3️⃣ Whisper AI transcreve o discurso original
4️⃣ Tradução automática para o idioma alvo
5️⃣ Edge-TTS gera nova voz no idioma alvo
6️⃣ MuseTalk 1.5 sincroniza lábios com novo áudio (qualidade HeyGen)
7️⃣ Download do vídeo traduzido pronto

Tempo estimado: 30-90s dependendo do tamanho do vídeo.

🤗 HuggingFace Spaces ZeroGPU — 100% GRÁTIS

Usa o Space vinthony/SadTalker hospedado no HuggingFace. GPU A100 gratuita. Sem limite de clientes. Sem conta necessária.

Obtenha em replicate.com → Account → API Tokens. Custo: ~R$0,25-R$0,75 por vídeo.

Como funciona: Você roda o servidor AvatarPilot Pro num VPS barato. Os 300 clientes acessam via navegador. O processamento pesado vai para a GPU A100 na nuvem (HuggingFace grátis ou Replicate pago).
